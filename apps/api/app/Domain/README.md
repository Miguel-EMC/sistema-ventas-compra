# Domain Layer

Esta carpeta agrupa el dominio de negocio de la nueva API:

- `Auth`
- `Users`
- `Catalog`
- `Assets`
- `Sales`
- `Customers`
- `Suppliers`
- `Cash`
- `Purchases`
- `Billing`
- `Reports`
- `Settings`

Cada subdominio debera contener entidades, reglas, repositorios del dominio y servicios de negocio que no dependan del framework.
